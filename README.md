## Usage

```bash
git clone ...
```
Go to each folder and use:
```bash
docker-compose up -d 
```
And don't forget add necessary **.env** file to specific folder.

For adding all **docker-compose.yml** files to commit use:
```bash
./gitadd.sh
```
